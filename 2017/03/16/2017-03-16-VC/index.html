退出才是投资的本质

现在做VC的人首要考虑的不是发展空间、回报周期、收益率等问题，
而是在他衰落前退出。

微软离破产只有18个月

不仅企业要有危机感，做VC的更要有。
本质上学习也是种投资，建模写代码？诗词歌赋？还是去跑跳坐打？
这都是个人时间的投资，而回报率却良莠不齐。
现今这些方向我都会浅尝却不知何时辄止，又不知该把主力投入哪个方向。

将有限的时间奉献给无限的为人民服务的共产主义事业中！

与其说雷锋精神是种思想，不如说是种信念。
我就是缺这样的信念啊。
